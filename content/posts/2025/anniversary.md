# 三年孤旅：从 0 到 33 枚 Merged PRs，我的 GitHub 演进全史

## 01. 序言：邮政绿与代码绿的重逢

2026 年 7 月 8 日，柳州的夏雨初晴。

就在刚才，我拆开了那封中国邮政 EMS 录取通知书专递。**广西柳州城市职业学院，计算机网络工程专业。** 当我抚摸着通知书上烫金的文字时，我的另一只手下意识地刷新了 GitHub 主页。那个绿色的贡献矩阵（Contribution Graph）在屏幕上闪烁，334 个绿色的方格像是一串密电码，记录了我过去三年来每一个深夜的挣扎与突破。

如果说录取通知书是我现实世界的通行证，那么这三年的 GitHub 提交记录，就是我灵魂深处的编年史。

从 2023 年初出茅庐的“脚本小子”，到 2026 年拥有 33 枚高质量 Merged PRs 的开源贡献者，我走过了一段极度孤独却又异常灿烂的旅程。这篇文章，我想写给那个在旧笔记本前敲下第一行 HTML 的少年，也写给每一个正在代码世界里逆流而上的开发者。

---

## 02. 2023：启蒙时刻——那一抹“邮政绿”之前的“入门蓝”

三年前的今天，我的 GitHub 主页还是一片荒芜。那时候的我，对“开源”的理解仅仅停留在“去别人的仓库下载免费的代码”。

### 2.1 恩师引路：Web 世界的第一道光
幸运的是，在那个懵懂的起点，我遇到了一位带我真正“入行”的恩师。

那不是枯燥的课堂教学，而是一场场关于“互联网如何运行”的实战演习。他教我不只是如何写 HTML 标签，更多的是教我如何选购服务器配置、如何处理复杂的运维逻辑、如何让一个静态页面在 Linux 环境下优雅地跑起来。

### 2.2 硬件的“青铜时代”：那台 4 核 8G 的旧华硕
那时候，我的“武器库”还非常寒酸。高一那一整年，我守着一台好几年前的旧华硕笔记本。4 核 CPU、8G 内存。

---

## 03. 2024：扩张之路——硬件迭代与“忙里偷闲”的助理生涯

进入 2024 年，我的 GitHub 贡献矩阵开始出现明显的深绿色块。这一年，我的生活节奏发生了剧烈的变化，技术栈也迎来了爆发。

### 3.1 武器升级：七彩虹将星 X15 的“火力全开”
高二那年，我终于迎来了我的“性能怪兽”——**七彩虹将星 X15 XS 22**。
原装的 i7-12700H 给了我处理大规模数据的底气。为了追求极致的多任务处理，我亲手将内存升级到了 **32GB**，并加装了一块 **2TB 的七彩虹“橘猫”硬盘**。

---

## 04. 2025：深耕之年——在 33 枚 Merged PRs 里的成长与碰撞

### 4.1 只有“大型项目”才能带来的快感
对我而言，只有像 **aeon-toolkit** 这样具有国际影响力的、代码逻辑严密到近乎苛刻的大型项目，当我的代码被其维护者认可并 Merge 的那一刻，那种战胜自我的快感才足以覆盖所有的疲惫。

---

## 05. 网络基因：从路由器到小型 NAS

### 5.1 家族传承的技术直觉
我从没有过“把家里网络搞瘫痪被父母吐槽”的尴尬经历，因为在我的家里，这种探索是被鼓励的。那种对网络拓扑的直觉，仿佛流淌在血脉里。
